Пример создания комиссии по проверке знаний (аттестационной)

В данной статье рассмотрим пример создания аттестационной комиссии по проверке знаний требований промышленной безопасности при эксплуатации сетей газораспределения и газопотребления (Б.7.1).

1. Для создания комиссии в первую очередь необходимо аттестовать в Ростехнадзоре не менее 3 специалиста из числа ИТР (1 – председатель комиссии из числа руководителей отделов, управлений и т.д., 2 члена комиссии).

2. После получения протокола аттестации подготовить приказ о создании комиссии (пример приказа во вложении, внизу страницы).

3. Также необходимо разработать программу подготовки к аттестации специалистов. По моему опыту, лучше ее утвердить приказом.

Программа включает в себя перечень законодательно-нормативных актов, план подготовки, экзаменационные вопросы, а также ключи к тестам.

4. По программе необходимо подготовить тестируемых, при необходимости провести консультации. Тестирование можно проводить устно, письменно, а можно воспользоваться сайтом.

5. Результат тестирования оформляется протоколом, при необходимости можно оформить удостоверение.
